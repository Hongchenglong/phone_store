## Phone Store
基于移动端的手机商城
基于SpringBoot + Vue的前后端分离开发架构

### 前端技术栈
Vue + Vant UI + less

Vant UI 安装命令
https://vant-contrib.gitee.io/vant/#/zh-CN/home
`cnpm i vant -S`
less 安装命令
`cnpm install less less-loader --save`
